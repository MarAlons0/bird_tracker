Bird Tracker — Help

Bird Tracker shows recent bird sightings in your area using eBird data and Claude AI to summarize what's been spotted.

How to use

  1. Tap Use My Current Location or search for a place on the Map tab.
  2. Select a search radius and the map will load nearby sightings.
  3. Tap any marker to see the bird name, location, and date.
  4. Go to AI Analysis for a summary of what's been seen in your area.
  5. Chat with the AI assistant to ask specific questions about local birds.

Tips

  • Use My Current Location for the fastest start.
  • Tap the expand icon on the map for fullscreen view.
